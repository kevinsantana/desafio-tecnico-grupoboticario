Bem vindo à documentação do Cashback Api!
=========================================
Cashback é um sistema para seus revendedores(as) cadastrarem suas compras e acompanhar o retorno de cashback de cada um.


Introdução
===========
Cashback quer dizer “dinheiro de volta”, e funciona de forma simples: o revendedor faz uma compra e seu benefício vem com a
devolução de parte do dinheiro gasto no mês seguinte.
